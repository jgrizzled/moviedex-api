# moviedex-api

**GET /movie**

Responds with list of movies

Headers:

&nbsp;&nbsp;&nbsp;Authorization: Bearer TOKEN

Query params: genre, country, avg_score

Live demo: https://rocky-stream-77018.herokuapp.com/

## install

`yarn`

## run

`yarn start`

## run with auto-restart on file change

`yarn dev`

## test

`yarn test`
